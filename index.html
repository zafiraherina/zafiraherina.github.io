Labs

Lab 1 – User Interface (UI)

Fundamental Android user interface development using XML layouts and Android Studio design tools.

I learned the fundamentals of Android user interface design using XML layouts. This lab helped me understand how to arrange components effectively and create a user-friendly interface. I also gained experience using Android Studio's design tools to build responsive layouts.

Lab 2 – Interaction

Implemented user interaction through buttons, input fields, and event handling in Android applications.

This lab introduced user interaction through buttons, input fields, and event handling. I learned how to capture user input and respond to actions using Java. The activity strengthened my understanding of Android application logic and user engagement.

Lab 3 – Material Design

Applied Material Design principles and modern UI components to improve usability and accessibility.

Through this lab, I explored Material Design principles and implemented modern UI components. I learned how consistent design improves usability and user experience. The lab also helped me understand the importance of visual hierarchy and accessibility.

Lab 4 – Navigation

Implemented multi-screen navigation and improved application flow using Android navigation concepts.

I learned how to implement navigation between multiple screens using Android navigation components. This lab improved my understanding of activity transitions and application flow. It also highlighted the importance of organizing content efficiently within an app.

Lab 5 – Room Database

Implemented local data storage using Room Database and CRUD operations.

This lab introduced Room Database for local data storage. I learned how to perform CRUD operations and manage persistent data within Android applications. Understanding Room helped me appreciate the importance of structured data management in mobile development.
